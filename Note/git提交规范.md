# git 提交规范

Angular 提交规范

```md
<type>(<scope>): <subject>

<body>

<footer>
```

```zsh
feat(user): 添加用户登录功能
```

| 类型       | 说明                               |
| ---------- | ---------------------------------- |
| `feat`     | 新功能                             |
| `fix`      | 修复 bug                           |
| `docs`     | 文档变更                           |
| `style`    | 代码格式（不影响代码运行）         |
| `refactor` | 重构（既不是新功能也不是修复 bug） |
| `perf`     | 性能优化                           |
| `test`     | 添加/修改测试                      |
| `chore`    | 构建过程或辅助工具变更             |
| `revert`   | 回滚                               |
| `build`    | 打包构建相关                       |
| `ci`       | CI 配置相关                        |
