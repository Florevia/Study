# CSS 选择器

## 一、基础选择器

| 选择器     | 示例            | 选中元素             |
| ---------- | --------------- | -------------------- |
| 通用选择器 | `*`             | 所有元素             |
| 元素选择器 | `div`           | 所有 `<div>`         |
| 类选择器   | `.btn`          | 所有 `class="btn"`   |
| ID 选择器  | `#header`       | `id="header"` 的元素 |
| 属性选择器 | `[type="text"]` | 带有该属性的元素     |

---

## 二、组合选择器（关系选择器）

| 选择器     | 示例      | 含义                     |
| ---------- | --------- | ------------------------ |
| 后代选择器 | `div p`   | `div` 内部所有层级的 `p` |
| 子代选择器 | `div > p` | `div` 的直接子元素 `p`   |
| 相邻兄弟   | `div + p` | 紧跟在 `div` 后面的 `p`  |
| 通用兄弟   | `div ~ p` | `div` 后面所有同级的 `p` |

---

## 三、属性选择器

| 选择器             | 含义                                |
| ------------------ | ----------------------------------- |
| `[attr]`           | 有该属性                            |
| `[attr="value"]`   | 属性值完全等于 value                |
| `[attr~="value"]`  | 属性值包含 value（空格分隔的词）    |
| `[attr\|="value"]` | 属性值等于 value 或以 `value-` 开头 |
| `[attr^="value"]`  | 属性值以 value 开头                 |
| `[attr$="value"]`  | 属性值以 value 结尾                 |
| `[attr*="value"]`  | 属性值包含 value                    |

---

## 四、伪类选择器

### 状态伪类

| 选择器      | 含义                       |
| ----------- | -------------------------- |
| `:hover`    | 鼠标悬停                   |
| `:active`   | 点击激活时                 |
| `:focus`    | 获得焦点                   |
| `:visited`  | 已访问的链接               |
| `:disabled` | 禁用状态                   |
| `:checked`  | 选中状态（checkbox/radio） |

### 结构伪类

| 选择器             | 含义             |
| ------------------ | ---------------- |
| `:first-child`     | 第一个子元素     |
| `:last-child`      | 最后一个子元素   |
| `:nth-child(n)`    | 第 n 个子元素    |
| `:nth-child(odd)`  | 奇数子元素       |
| `:nth-child(even)` | 偶数子元素       |
| `:nth-child(2n+1)` | 奇数（公式写法） |
| `:only-child`      | 唯一子元素       |
| `:first-of-type`   | 同类型的第一个   |
| `:last-of-type`    | 同类型的最后一个 |
| `:nth-of-type(n)`  | 同类型的第 n 个  |
| `:empty`           | 没有子节点的元素 |
| `:not(selector)`   | 排除某些元素     |

---

## 五、伪元素选择器

| 选择器           | 含义                 |
| ---------------- | -------------------- |
| `::before`       | 元素内容之前插入内容 |
| `::after`        | 元素内容之后插入内容 |
| `::first-line`   | 第一行文本           |
| `::first-letter` | 首字母               |
| `::selection`    | 被选中的文本         |
| `::placeholder`  | 输入框占位符         |

---

## 六、选择器优先级（权重）

从高到低：

| 权重 | 选择器类型   | 示例                     |
| ---- | ------------ | ------------------------ |
| ∞    | `!important` | `color: red !important;` |
| 1000 | 内联样式     | `style="color: red"`     |
| 100  | ID 选择器    | `#header`                |
| 10   | 类/属性/伪类 | `.btn` `[type]` `:hover` |
| 1    | 元素/伪元素  | `div` `::before`         |
| 0    | 通用选择器   | `*`                      |
