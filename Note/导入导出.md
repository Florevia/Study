# 导入导出

## 分类
 
### 默认导出

```js
// 导出
export default {
  name: 'Vue',
  version: '3.0.0'
}
// 导入
import Vue from './vue.js';
```

### 命名导出

```js
// 导出
export const name = 'Vue';
export const version = '3.0.0';
// 导入
import { name, version } from './vue.js';
```

### 混合导出

```js
// 导出
export default {
  name: 'Vue',
  version: '3.0.0'
}
export const name = 'Vue';
export const version = '3.0.0';
// 导入
import Vue, { name, version } from './vue.js';
```

## 注意事项

- 默认导出只能有一个，命名导出可以有多个
- 导入时可以使用默认导出的变量名（使用任意名称，不需要花括号），也可以使用命名导出的变量名（必须使用确切名称，需要花括号）
- 混合导出时，默认导出的变量名必须在最前面（默认导出在前，命名导出在后）
