### 包管理器（package manager）

是一类软件工具，用来查找、安装、升级、配置和移除软件包（package）。

它能自动处理依赖关系（一个软件运行所需的其它库或包）、从远程仓库下载包、校验完整性，并能把已装软件记录在本地数据库里，方便管理与回滚。

### 包管理器常见功能

- 从远程仓库（repository / registry）搜索并下载安装包。
- 自动解决并安装依赖（dependency resolution）。
- 升级、卸载、回滚软件包。
- 验证包签名与完整性。
- 本地缓存与锁定（lockfile）以保证可复现构建。
- 发布/维护包到远程仓库（对包作者）。
