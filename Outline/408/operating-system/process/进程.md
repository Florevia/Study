# 进程

## 什么是进程？（what）

进程=正在运行的程序+所需资源和状态（进程控制块）

## 内存中的进程

- 代码段
- 数据段
- bss段
- heap
- stack

## 进程控制块 process control block

- process ID
- process status
  - 就绪
  - 运行
  - 阻塞
  - 终止
  - 僵尸
- 打开的文件（文件描述符）
- 属于哪个用户

## 进程间通信（Inter-process communication）

- 管道
- 消息队列
- 共享内存
- 信号
- socket（套接字）

- [x] SIGINT, SIGTERM, SIGKILL 三者差异
