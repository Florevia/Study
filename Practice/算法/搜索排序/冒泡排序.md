# 冒泡排序

## 算法思想

越小的元素会经由交换慢慢“浮”到数列的顶端，就像水中的气泡一样逐渐上升至表面。

## 示例

```js
Array.prototype.bubbleSort = function () {
  for (let j = 0; j < this.length - 1; j++) {
    for (let i = 0; i < this.length - 1; i++) {
      if (this[i] > this[i + 1]) {
        const temp = this[i];
        this[i] = this[i + 1];
        this[i + 1] = temp;
      }
    }
  }
};

const arr = [3, 2, 1];
arr.bubbleSort();
console.log(arr);
```
