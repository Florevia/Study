# 堆

- 一种特殊的完全二叉树数据结构
- 父节点与子节点之间有特定的大小关系
  - 最小堆：父节点的值小于等于子节点的值
  - 最大堆：父节点的值大于等于子节点的值

### 索引公式：

- 父节点：(i - 1) / 2
- 左子节点：2 \* i + 1
- 右子节点：2 \* i + 2

    let small = index;
    if (

      this.heap[leftIndex] < this.heap[small]

    if (small !== index) {

      this.shiftDown(small);
    }

  // 插入
  insert(value) {
    this.heap.push(value);
    this.shiftUp(this.heap.length - 1);
  }
  // 删除堆顶
  pop() {
    if (this.heap.length === 0) return undefined; // 空堆
    if (this.heap.length === 1) return this.heap.pop(); // 只有一个元素

    const top = this.heap[0]; // 保存堆顶
    this.heap[0] = this.heap.pop(); // 末尾移到堆顶
    this.shiftDown(0); // 下沉调整
    return top; // 返回被删除的值
  }
  // 获取堆顶
  peek() {
    return this.heap[0];
  }
  // 获取堆的大小
  size() {
    return this.heap.length;
